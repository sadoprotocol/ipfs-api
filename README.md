# ipfs-api

## Deployment

### Requirement

- MongoDB

<br />

### Configuration

Copy `dotenv` to `.env`.\
Edit `.env` and set proper parameters.

<br />

Install packages:

```javascript
$ npm install
```

<br />

Start:

```javascript
$ npm start
```

<br />
<br />

## Usage

Examples of html form and upload javascript procedure are available in [examples](https://github.com/cakespecial/ipfs-api/tree/main/examples).
