# ipfs-api

## Deployment

### Requirement

- MongoDB

### Configuration

Copy `dotenv` to `.env`.
Edit `.env` and set proper parameters.

<br>

Install packages:
`$ npm install`

<br>

Start:
`$ npm start`

## Usage

Examples of html form and upload javascript procedure are available in `/examples`
