## Deployment

### Requirement

- MongoDB

<br />

### Configuration

Copy `dotenv` to `.env`.
Edit `.env` and set proper parameters.

<br />

Install packages:
`$ npm install`

<br />

Start:
`$ npm start`

<br />
<br />
